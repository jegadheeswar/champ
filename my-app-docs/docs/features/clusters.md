# Clusters

Manage Kubernetes clusters across cloud providers from a single interface.

## Supported Platforms

- Amazon EKS
- Google GKE
- Azure AKS
- Self-managed Kubernetes

## Creating a Cluster

1. Go to **Clusters**
2. Click **+ New Cluster**
3. Choose provider and region
4. Configure node size, count, and network settings
5. Click **Create**

## Monitoring

View real-time metrics:
- CPU/Memory usage
- Pod status
- Node health

## Scaling

- **Manual**: Adjust node count in cluster settings
- **Auto**: Enable autoscaling based on load
