# Tools

Integrate popular DevOps tools like Argo CD, Jenkins, and more.

## Supported Tools

- Argo CD (GitOps)
- Jenkins (CI/CD)
- Terraform (Infrastructure as Code)
- Helm (Package Manager)

## Adding a Tool

1. Go to **Tools**
2. Click **+ Add Tool**
3. Select tool and enter connection details
4. Click **Connect**
