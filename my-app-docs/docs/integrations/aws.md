# AWS Integration

Connect your AWS account to manage EC2, S3, RDS, and more from CloudMaSa.

## Setup Steps

1. Go to **Work Flow > AWS**
2. Click **+ Add Account**
3. Enter your **AWS Access Key ID** and **Secret Access Key**
4. Select your **Region** (e.g., `us-east-1`)
5. Click **Test Connection**

## Supported Services

- EC2 (Virtual Machines)
- S3 (Storage)
- RDS (Databases)
- Lambda (Serverless)
- VPC (Networking)
