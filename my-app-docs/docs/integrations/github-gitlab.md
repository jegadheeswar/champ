# GitHub & GitLab Integration

Connect your repositories to trigger workflows, manage branches, and deploy code.

## Connecting GitHub

1. Go to **SCM Connector > GitHub**
2. Click **+ Add Repository**
3. Authorize CloudMaSa
4. Select repo and branch
