# Prometheus & Grafana Integration

Monitor your applications and infrastructure with built-in dashboards.

## Setup

1. Go to **Integrations > Prometheus/Grafana**
2. Click **+ Add Monitoring**
3. Enter Prometheus endpoint (e.g., `http://prometheus:9090`)
4. Click **Connect**

## Dashboards

Pre-built dashboards for:
- Cluster Health
- Pod Performance
- Network Traffic
- Database Metrics

## Alerts

Set alerts:
- CPU over 80%
- Memory leak detected
- HTTP 5xx errors
