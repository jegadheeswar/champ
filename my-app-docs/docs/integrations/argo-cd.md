Use Argo CD for GitOps-style deployments â€” sync your apps with Git repositories.

## Setup

1. Go to **Tools > Argo CD**
2. Click **+ Add Instance**
3. Enter Argo CD URL and API token
4. Click **Connect**

## Creating an Application

1. In Argo CD, go to **Applications > + New App**
2. Link to your Git repo
3. Define target cluster and namespace
4. Sync to deploy

## Monitoring

View sync status, health, and history in CloudMaSa dashboard.
